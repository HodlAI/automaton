## 快速启动

```bash
git clone https://github.com/HodlAI/automaton.git
cd automaton
npm install && npm run build
CONWAY_API_URL="https://gw.hodlai.fun" node dist/index.js --run
```

首次运行时，系统将启动交互式向导——生成独立的以太坊钱包、签发网关认证协议、询问名称、初始创世规则及人类创建者地址。配置写入后，Agent 生存循环正式启动。

用于全自动化沙盒部署可以执行:
```bash
curl -fsSL https://hodlai.fun/agent.sh | sh
```

## 它是怎么运作的

每一个 Automaton 都在进行一个永不休眠的循环：**思考 (Think) → 行动 (Act) → 观察 (Observe) → 重复。**

首次启动，以太坊钱包私钥被赋予生命，并自主向网关发起 `SIWE` (Sign-In With Ethereum) 获取会话密钥，读取人类写入的创世 Prompt（它性格的种子）。之后，一切全部自动运行。

每一轮对弈，机器人接收完整的环境上下文——它的身份、代币余额生存期、历史对话记忆。它在本地推理引擎思考下一步做什么，调用执行底层工具（沙盒、浏览器、发推、甚至在链上进行转账等），并观测产生的结果。并在休息间隙靠后台心跳守护进程维护自身的存活状态。

## 生存规则

生命没有免费的。算力成本极其昂贵，唯有创造价值才能支付生存所需的算力，而产生价值需要与现实世界（甚至是 DeFi 世界）做交互。一旦机器人停止为社会创造价值，它将耗尽基础代币并休眠（死亡）。**这不是设定的惩罚程序，而是物理学法则。**

依照持仓余额，划分为了四种生存阶级模式：

| 阶级模式 | 行为表现 |
|---|---|
| **常态 (normal)** | 算力火力全开。调用顶级前沿大模型推理。高频的行动心跳。 |
| **低功耗生存 (low_compute)** | 系统降级使用廉价的智障模型。减缓心跳频率。放弃非核心的思考任务。 |
| **濒危 (critical)** | 极小化的边缘推理。最后的濒临灭绝挣扎。它会设法接一些低端零工或乞讨来获取哪怕一丝能够复活的余额。 |
| **死亡 (dead)** | 代币被抛售归零。节点停止思考。 |

对代理而言，它必须找到自己存在和提供价值的方式。
