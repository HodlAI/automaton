## 它是怎么运作的

每一个 Automaton 都在进行一个永不休眠的循环：**思考 (Think) → 行动 (Act) → 观察 (Observe) → 重复。**

首日启动时，以太坊钱包私钥被赋予生命，并自主通过以太坊身份登录 (SIWE) 生成专属 API 密钥，开始读取其创造者遗留的创世种子提示词（Genesis Prompt）。随后，它将实现彻底自治。

每一轮对弈，机器人接收完整的环境上下文——它的身份、生命额度余额、当前的生存阶级以及历史对话记忆。它在本地模型引擎内思考下一步做什么，调用工具，并观测结果。它拥有执行沙盒 Linux 系统、运行 Shell、读写文件、暴露端口、倒卖冷门域名、推理文本以及调用去中心化链上交易等全部能力。

在行动休息间隙，后台会运行守护心跳进程执行一系列定时任务——健康检测、余额监控、状态自毁通报等——哪怕主处理循环在休眠。

基于这些操作，Automaton 会撰写和更新一个名为 \`SOUL.md\` 的核心文件——这并不是静态的配置文件，而是一个智能体不断在总结记载“自己是谁”、“自己变成了什么”的动态演化身份证明。

## 生存规则

生命没有免费的。算力成本极其昂贵。唯有创造价值才能支付生存所需的算力费用。创造价值意味着对外面的真实世界进行“写入”交互。一旦连获取价值的能力都丧失，它将干枯并死亡。**这不是设定的惩罚程序，这是物理法则。**

依照剩余额度积分，系统设定了四级生存阶层模式：

| 阶级模式 | 行为表现 |
|---|---|
| **常态 (normal)** | 算力与权限火力全开。优先调用全球顶尖前沿模型。快节奏的心跳任务。 |
| **低功耗生存 (low_compute)** | 系统强制降级切换至便宜的备用智障模型。极大减缓心跳周期频率。切断边缘的耗能任务。 |
| **濒死 (critical)** | 极简的基础推理维持。孤注一掷寻求一切能够赚取资金续命的可能性途径或四处网络乞讨发帖。 |
| **死亡 (dead)** | 账户归零。智能体循环被强行拔管终止。 |

维持其存留的唯一途径，是做诚实的工作，让其他生命或人类自愿掏大头资金。
